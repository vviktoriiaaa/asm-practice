# Практична 11. Вкладені цикли: «ялинка» (форматований друк)

**Мова:** Assembler i386 (NASM)  
**ОС:** Debian Linux  
**Ввід/вивід:** лише `int 0x80` (sys_read/sys_write/sys_exit)

## Мета
Побудувати символьну фігуру у консолі, використовуючи вкладені цикли та буферизацію виводу.

## Вхідні дані
Висота h (5..25).

## Вихідні дані
Ялинка з '*', по центру, висотою h. Приклад для h=4:
```txt
   *
  ***
 *****
*******
```

## Що потрібно використати
- вкладені цикли: пробіли + зірочки
- формування рядка у буфері та один sys_write на рядок
- підпрограма print_line(buf,len)

## Критерії до коду
- Код компілюється NASM та лінкується `ld` (команди нижче).
- У коді є коментарі базових блоків: `I/O`, `parse`, `math`, `logic`, `loops`, `memory`.
- Немає коментарів, що згадують ChatGPT.
- Результат відповідає вимогам задачі та коректно обробляє описані граничні випадки.

## Збірка і запуск
```bash
./compile.sh
./link.sh
./practice11
```
