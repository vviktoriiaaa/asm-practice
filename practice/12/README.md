# Практична 12. Рядки в пам’яті: пошук підрядка

**Мова:** Assembler i386 (NASM)  
**ОС:** Debian Linux  
**Ввід/вивід:** лише `int 0x80` (sys_read/sys_write/sys_exit)

## Мета
Реалізувати пошук підрядка в рядку та підрахунок кількості входжень.

## Вхідні дані
Два рядки: text (до 200), pattern (до 50).

## Вихідні дані
1) Перша позиція входження або -1
2) Кількість входжень (без перекриття)

## Що потрібно використати
- буфери у .bss
- власна strlen
- наївний пошук (дві вкладені перевірки)
- обробка pattern == ""

## Критерії до коду
- Код компілюється NASM та лінкується `ld` (команди нижче).
- У коді є коментарі базових блоків: `I/O`, `parse`, `math`, `logic`, `loops`, `memory`.
- Немає коментарів, що згадують ChatGPT.
- Результат відповідає вимогам задачі та коректно обробляє описані граничні випадки.

## Збірка і запуск
```bash
./compile.sh
./link.sh
./practice12
```
