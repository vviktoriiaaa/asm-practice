# Практична 14. Сортування: selection sort + медіана

**Мова:** Assembler i386 (NASM)  
**ОС:** Debian Linux  
**Ввід/вивід:** лише `int 0x80` (sys_read/sys_write/sys_exit)

## Мета
Виконати сортування масиву та обчислити медіану.

## Вхідні дані
n (10..100), далі n чисел.

## Вихідні дані
1) Масив до сортування
2) Масив після сортування (за зростанням)
3) Медіана (для парного n — нижня середня)

## Що потрібно використати
- selection sort
- обмін двох dd
- вкладені цикли i/j
- уважна індексація

## Критерії до коду
- Код компілюється NASM та лінкується `ld` (команди нижче).
- У коді є коментарі базових блоків: `I/O`, `parse`, `math`, `logic`, `loops`, `memory`.
- Немає коментарів, що згадують ChatGPT.
- Результат відповідає вимогам задачі та коректно обробляє описані граничні випадки.

## Збірка і запуск
```bash
./compile.sh
./link.sh
./practice14
```
