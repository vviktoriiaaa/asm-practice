# Практична 4. Ввід числа і базові операції з регістрами та пам’яттю

**Мова:** Assembler i386 (NASM)
**ОС:** Debian Linux
**Ввід/вивід:** лише `int 0x80` (sys_read/sys_write/sys_exit)

## Вхідні дані
Прочитати з консолі рядок
сконвертувати його в число тиау int і покласти в регістр AX
використати код з попередньоїх практичної для конвертації числа в регістрі AX

## Вихідні дані
те саме чило надруковане на консолі

## Критерії до коду
- Код компілюється NASM та лінкується `ld` (команди нижче).
- У коді є коментарі базових блоків: `I/O`, `parse`, `math`, `logic`, `loops`, `memory`.
- Немає коментарів, що згадують ChatGPT.
- Результат відповідає вимогам задачі та коректно обробляє описані граничні випадки.

## Збірка і запуск
```bash./compile.sh
./link.sh
./practice4
```
