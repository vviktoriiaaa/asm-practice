# Практична 5. Сума цифр і робота з діленням (unsigned)

**Мова:** Assembler i386 (NASM)  
**ОС:** Debian Linux  
**Ввід/вивід:** лише `int 0x80` (sys_read/sys_write/sys_exit)

## Мета
Закріпити unsigned-ділення div, роботу із залишком та прості цикли.

## Вхідні дані
Одне додатне число x (1..2_000_000_000).

## Вихідні дані
Два рядки:
1) sumDigits(x)
2) len(x)
Третій рядок (додатково): sumDigits(x) + K

## Що потрібно використати
- div (unsigned): EDX:EAX / r32
- цикл while x > 0
- EDX обнуляти перед div
- підпрограми atoi/itoa (можна повторно використати з попередньої практики)

## Критерії до коду
- Код компілюється NASM та лінкується `ld` (команди нижче).
- У коді є коментарі базових блоків: `I/O`, `parse`, `math`, `logic`, `loops`, `memory`.
- Немає коментарів, що згадують ChatGPT.
- Результат відповідає вимогам задачі та коректно обробляє описані граничні випадки.
