# Практична 8. Пошук даних: лінійний пошук + підрахунок входжень

**Мова:** Assembler i386 (NASM)  
**ОС:** Debian Linux  
**Ввід/вивід:** лише `int 0x80` (sys_read/sys_write/sys_exit)

## Мета
Реалізувати пошук значення по масиву та зібрати статистику входжень.

## Вхідні дані
n (10..100), далі n чисел, далі target.

## Вихідні дані
1) Перший індекс target або -1
2) Кількість входжень
3) Список всіх індексів через пробіл (або порожній рядок, якщо немає)

## Що потрібно використати
- масив у .bss
- лінійний пошук
- форматований друк списку індексів
- акуратна обробка випадку «не знайдено»

## Критерії до коду
- Код компілюється NASM та лінкується `ld` (команди нижче).
- У коді є коментарі базових блоків: `I/O`, `parse`, `math`, `logic`, `loops`, `memory`.
- Немає коментарів, що згадують ChatGPT.
- Результат відповідає вимогам задачі та коректно обробляє описані граничні випадки.

## Збірка і запуск
```bash
./compile.sh
./link.sh
./practice8
```
